# Repository Overview

> Auto-generated by Laravel Generator Command

## Purpose

This application scans Laravel codebases and automatically generates `.llm.md` files that summarize the project for use by large language models. It is primarily intended for developers working with Laravel projects who want to give LLMs structured context about their code. It solves the problem of manually assembling high-level project overviews by compiling tech stack details, README summaries, directory structure, routes, and test-derived features into a single file.

## Tech Stack

Laravel 12, PHP 8.2
