## Purpose

This application generates `.llm.md` files that summarize a Laravel project’s tech stack, structure, routes, and features so large language models can better understand the codebase. It is primarily aimed at developers working on Laravel projects. It solves the problem of providing concise, structured context to LLMs without manually curating project documentation.
